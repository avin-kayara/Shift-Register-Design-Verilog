# Shift-Register-Design-Verilog
Designed an 8-bit Shift Register in Verilog with two modules: SIPO (Serial-In Parallel-Out) and PISO (Parallel-In Serial-Out), each supporting both Shift-Left and Shift-Right operations controlled by a shift_dir signal. Simulated using Icarus Verilog and verified with GTKWave. All 4 modes work correctly as validated by the testbench.
